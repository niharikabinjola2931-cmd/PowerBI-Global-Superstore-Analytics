# Global Superstore Sales & Profit Analytics — Power BI

## Overview
Interactive Power BI dashboard analyzing global 
superstore sales data across regions, segments 
and categories.

## Data Model
Star schema with 3 tables:
- Orders (Fact table)
- Customers (Dimension)
- Products (Dimension)

## KPIs
- Total Sales: 5.63M
- Total Customers: 4.87K
- Total Profit: 478.93K
- Total Orders: 25K
- Total Quantity: 88K
- Total Shipping Cost: 606.71K
- Total Products: 3.691K

## Key Insights
- California generated highest profit (42.31K)
- Consumer segment = 51.53% of total sales
- Chairs top sub-category (705.69K in sales)
- Office Supplies highest quantity sold
- West region most profitable

## Tools
Power BI · DAX · Power Query · Star Schema · ETL

## Dataset
Global Superstore Dataset — Kaggle
